# AI Garden Assistant 🌱

AI Garden Assistant is a Streamlit-based application that helps users identify plant problems and provides smart recommendations.

## Features
- Plant Disease Suggestions
- Plant Image Upload
- Plant Health Score
- Plant Database
- Care Recommendations
- Daily Plant Checklist

## Tech Stack
- Python
- Streamlit

## Future Scope
- AI Image Recognition
- Weather API
- Fertilizer Recommendation
- Mobile App
